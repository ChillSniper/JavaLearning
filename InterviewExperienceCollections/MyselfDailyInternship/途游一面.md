# 途游后端一面

## 面试内容如下

- 自我介绍
- 你看你这里面主要是学的 Java 是吧？对 Python、Go 这些有了解过吗？

### 项目考察

- 我看你校园生活服务平台是你们学校里面的课题，还是网上的？
- 这个里面你用到分布式锁，你可以具体说一下具体的原理是什么？

### 八股杂项

- Redis 除了分布式锁，还有哪些可以使用的数据结构？
- MySQL 的话有几种索引？
- 日常在使用索引建表的时候，需要注意哪些问题？
- 接口肯定需要做健全身份识别，常见的健全方案有哪些？你用的是什么？
- JWT Token 的结构是什么样子的？
- RocketMQ 你当时用的时候有去了解过它的原理吗？
- 了解过一致性哈希算法吗？
- 布隆过滤器了解过吗？可以举个例子吗？
- 非对称加密算法 RSA 了解过吧？举一下它在哪些场景用到了？
- Java 里面的双亲委派机制是什么样的？
- 了解过异步 IO（AIO）吗？
- 找实习是在北京吗？计划实习多长时间？
- 平时通过什么方式学习技术？感兴趣的方向是什么？
- 做一个分布式系统，整个架构体系是什么样子的？
- CAP 定理是什么？
- 对游戏感兴趣吗？
- 如果做一个发红包系统，会遇到哪些挑战？

- 面试后续流程
