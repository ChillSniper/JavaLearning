# 京东创新零售一面

## 面试内容

- 简单的自我介绍一下
- 本科是23年入学，正常是27年毕业是吧？就是毕业之后不打算再继续读研吗？这是出于什么考虑呢？

### OS

- 咱们提起来计算机操作系统，能举个例子吗？比如说咱们编写的这个代码程序，在操作系统上是怎么样被执行的，整个过程大概是什么样的？
- 刚才你提到进程，能够聊聊常见的进程调度策略算法有哪些吗？
- 你能讲讲这银行家算法是怎么样吗？大概的原理。
- 常见的IO模型有哪些？可以介绍一下吗？
- 计算机操作系统中还提到这个中断概念，能简单讲讲计算机为什么会产生中断吗？

### Redis

- 在Redis里边的话，你这个底层大概是怎么样一个实现原理？或者说Redis它为什么存取这么快？
- 能展开讲讲，IO多路复用是怎么样一个架构模型？

### MySQL

- 对MySQL底层的这个索引结构特征，能够简单的表述一下吗？
- 基于这样的一个索引结构特征，常见的SQL语句书写规范，或者说是这个语句调优优化的方式方法有哪些？

### 场景题

- 根据我的日历和你的日历快速匹配出来哪一天哪个时段咱们两个可以进行面试。你会怎么样选择数据结构和简单的算法？
- 跨银行转账服务，比如从工行扣10块，转到农行。你觉得里边会涉及到哪些关键的技术设计要点？
- 它是跨行的系统，数据库和服务肯定都部署在不同的服务器上。在这种场景下，如何保证一致性？
- 还有没有其他的风险？比如重复扣减，这种场景下怎么样规避？常见的幂等保持机制了解过吗？
- 讲一下对分布式事务的理解？

### 闲聊

- 你觉得这个学习过程中最大的收获，或者说当时遇到的最大的挑战问题是啥？
- 你自己个人对自己的这个职业发展这一块有没有过一些构思，或者说是角色定位上的一些设想？

### 反问

- 面试后续流程是常规的一轮技术面加一轮HR面吗？
- 面试结果大概几天之内能出？
- 关于后续技术栈的学习或改进建议？
